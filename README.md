# SDR Efficiency Cleanup – Project Tracker

Internal project tracker for Cribl's SDR Efficiency Cleanup initiative.

## Quick Start

1. Download `index.html`
2. Open it in Chrome
3. Start tracking tasks

## Team Sync (Supabase)

To enable real-time sync across the team:

1. Create a free project at [supabase.com](https://supabase.com)
2. Run the SQL from the setup panel in Supabase's SQL Editor
3. Click "LOCAL ONLY" in the tracker → enter your Project URL + anon key → Connect
4. Share the same credentials with teammates

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Import → select this repo
3. Click Deploy (no config needed — it's a static HTML file)
4. Share the Vercel URL with the team

## Features

- 100 tasks across 8 phases
- 17 editable success metrics
- Add / Edit / Delete tasks
- Filter by phase, owner, status, week
- Column sorting
- Real-time Supabase sync
- Cribl brand colors + Ian the Goat
